# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-15

### Added
- Core note management (create, read, update, delete)
- 6 beautiful note colors for visual categorization
- 5 note categories (Personal, Work, Ideas, Urgent, Other)
- Custom tags support for notes
- Favorite notes functionality
- Advanced search with multi-criteria filtering
- Sort options: Recent, Oldest, Alphabetical
- Smart date formatting (e.g., "5m ago", "Yesterday")
- Material 3 design system
- Smooth animations and transitions
- Masonry grid layout for notes
- Responsive design across all platforms
- Empty state with helpful guidance
- Undo functionality for deleted notes
- Dynamic text colors based on background

### Features
- Android support (6.0+)
- iOS support (11.0+)
- Web support (Chrome, Firefox, Safari)
- Windows support
- macOS support
- Linux support
